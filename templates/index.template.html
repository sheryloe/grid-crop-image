Core Features

핵심 기능

검색 키워드만 맞춘 페이지가 아니라 실제 사용자 작업 흐름에 맞는 기능을 요약했습니다.

클립보드 이미지 붙여넣기

`Ctrl + V`로 이미지 데이터를 바로 불러오고, 파일 복사 상황에서는 지원 이미지 자동 감지를 수행합니다.

다중 사각형 크롭

여러 개의 크롭 사각형을 생성, 이동, 삭제, 초기화할 수 있고 저장 순서도 유지됩니다.

JSON 설정 저장과 불러오기

원본 이미지 경로, 출력 폴더, 줌, 좌표, 설정 상태를 저장하고 비율 보정도 수행합니다.

확대, 축소, 맞춤 보기

Ctrl + 마우스휠 확대/축소, Shift + 마우스휠 가로 스크롤로 긴 이미지도 정밀하게 조정할 수 있습니다.

순차 파일 저장

확정한 사각형 순서대로 `rect_01`, `rect_02` 형식으로 저장하고 중복 파일명도 자동 회피합니다.
