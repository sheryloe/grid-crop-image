여러 문서(README, Service-Roadmap, docs/index.html, notion)에 흩어져 있던 향후 개발 계획을 종합하고, 현재 시점에서 유효한 TODO 항목들을 우선순위에 따라 재분류한 목록입니다.

## Priority 1: 핵심 기능 및 사용자 경험 강화

사용자의 편집 실수를 줄이고, 더 정교한 제어를 가능하게 하여 앱의 완성도를 높이는 데 중점을 둡니다.

- **실행 취소 / 다시 실행 (Undo/Redo)**
  - 사각형 생성, 이동, 크기 조절, 삭제 등 모든 편집 작업에 대한 실행 취소 및 다시 실행 기능을 추가합니다.
  - `Ctrl+Z`, `Ctrl+Y` 단축키를 지원합니다.

- **작업 세션 자동 저장 및 복원**
  - 예기치 않게 앱이 종료되었을 때, 다음 실행 시 마지막 작업 상태(이미지, 사각형 목록 등)를 복원할 수 있는 기능을 제공합니다.

- **사각형 목록 관리 패널**
  - 캔버스 옆에 현재 생성된 모든 사각형의 목록을 보여주는 패널을 추가합니다.
  - 목록에서 특정 사각형을 선택하면 캔버스에서도 해당 사각형이 선택되도록 연동합니다.
  - 목록의 순서를 드래그 앤 드롭으로 변경하여, 저장되는 파일(`_rect_01`, `_rect_02`...)의 순서를 직접 제어할 수 있게 합니다.
  - (확장) 목록에서 각 사각형의 이름을 직접 편집하고, 이 이름을 파일명에 반영하는 옵션을 제공합니다.

## Priority 2: 생산성 및 워크플로우 고도화

반복 작업을 더욱 줄이고, 전문적인 사용 시나리오를 지원하는 기능들입니다.

- **크롭 프리셋 저장 및 관리**
  - 자주 사용하는 그리드 설정(예: 2x2, 3x3)이나 특정 좌표들을 '프리셋'으로 저장하고, 메뉴에서 한 번에 불러올 수 있는 기능을 추가합니다.

- **OCR(광학 문자 인식) 보조 기능**
  - 이미지 내의 텍스트 블록을 자동으로 감지하여 사각형 영역을 추천해주는 기능을 검토합니다.
  - 잘라낸 이미지의 텍스트를 추출하여 파일명에 사용하거나 클립보드에 복사하는 기능을 검토합니다.

- **드래그 정렬 보조선 (Snapping Guides)**
  - 사각형을 이동하거나 크기를 조절할 때, 근처에 있는 다른 사각형의 경계나 중심선에 자석처럼 달라붙어 정렬을 쉽게 만듭니다.
