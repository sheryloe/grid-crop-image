Grid Crop Image는 한 장의 이미지를 여러 영역으로 나누어 저장해야 하는 작업을 위해 만든 Tkinter 기반 도구입니다. 스크린샷을 붙여넣고, 여러 크롭 영역을 지정하고, 같은 레이아웃을 JSON으로 다시 불러와 반복 작업 시간을 줄일 수 있습니다.

레이아웃을 한 번 잡아 두면 다음 작업이 빨라집니다

이미지 위에 필요한 영역을 미리 나눠 두고, 저장 위치와 설정 파일까지 이어서 관리할 수 있습니다.

01

이미지 열기와 클립보드 붙여넣기

파일을 직접 열거나 `Ctrl+V`로 캡처 이미지를 붙여 넣어 작업 시작 시간을 줄입니다.

02

여러 영역을 한 번에 정의

사각형을 여러 개 만들고 이동하면서 한 장의 원본 이미지에서 필요한 조각을 순차 저장할 수 있습니다.

03

반복 레이아웃은 JSON으로 저장

자주 쓰는 위치 조합을 JSON 파일로 남겨 두면 블로그 카드나 문서 캡처 같은 반복 작업이 빨라집니다.

Workflow

편집기가 아니라, 반복 크롭 작업에 맞춘 짧은 동선

이 앱은 레이어나 템플릿 편집보다 이미지를 빠르게 자르는 반복 작업을 위한 구조입니다. 이미지를 불러오고, 사각형을 지정하고, 그대로 분할 저장하는 데 필요한 버튼만 전면에 배치했습니다.

1. 원본 이미지 불러오기

파일 선택 또는 클립보드 붙여넣기로 작업 이미지를 바로 올립니다.

2. 크롭 사각형 배치

이미지 위에서 여러 영역을 지정하고, 번호와 외곽선으로 현재 선택 상태를 확인합니다.

3. 설정 저장과 분할 내보내기

출력 폴더를 고른 뒤 조각을 저장하고, 같은 좌표 구성을 JSON으로 다시 쓸 수 있습니다.

Runtime capture

실제 작업 화면에서 툴바와 선택 영역이 어떻게 보이는지 확인할 수 있습니다

아래 이미지는 로컬 실행 환경에서 직접 캡처한 화면입니다. 첫 이미지는 전체 워크스페이스, 두 번째는 사각형 선택이 어떤 방식으로 겹쳐 보이는지 확대해 보여 주는 상세 장면입니다.

STEP 1 전체 레이아웃을 한눈에 정리

툴바, 저장 위치, 배율, 분할 버튼이 모두 상단에 있어서 작업 중 위치 이동이 적습니다.

STEP 2 세부 영역을 빠르게 미세 조정

선택된 사각형은 강조되고, 번호가 붙어서 여러 영역을 다룰 때도 맥락을 잃지 않습니다.

FAQ

자주 묻는 질문

Grid Crop Image는 어떤 작업에 잘 맞나요?

블로그용 스크린샷 분할, UI 캡처 조각내기, SNS 카드 이미지를 반복 크롭하는 작업에 특히 잘 맞습니다.

여러 영역을 한 번에 저장할 수 있나요?

가능합니다. 여러 개의 사각형을 지정한 뒤 분할 시작 버튼으로 순차 저장할 수 있습니다.

같은 좌표 구성을 다시 불러올 수 있나요?

네. JSON 설정 저장과 불러오기를 통해 같은 레이아웃을 반복 작업에 재사용할 수 있습니다.
