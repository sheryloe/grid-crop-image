Windows Image Crop Tool Clipboard Paste JSON Layout Save

빠르게 자르고, 여러 조각으로 저장하는 Windows 이미지 도구

Grid Crop Image는 Windows에서 스크린샷 자르기, 클립보드 이미지 붙여넣기, 다중 영역 크롭, JSON 레이아웃 저장과 불러오기, 순차 저장을 지원하는 이미지 분할 도구입니다. 반복되는 스크린샷 자르기, 블로그용 이미지 분할, 동일한 레이아웃 재사용 작업을 빠르게 처리할 수 있습니다.

Repository Usage Guide
Ctrl + V 클립보드 이미지를 바로 불러와 파일 저장 없이 작업을 시작할 수 있습니다.
JSON Save 크롭 좌표와 작업 상태를 저장하고 다시 불러올 수 있습니다.
Ordered Export 선택한 순서대로 결과 파일을 정리해서 저장합니다.

Clipboard Paste

Multi-Region Crop

JSON Layout Save and Load

FAQ

자주 묻는 질문

프로젝트 목적과 핵심 사용 흐름을 빠르게 확인할 수 있도록 정리했습니다.

어떤 작업에 잘 맞나요?

스크린샷 분할, 블로그 자산 제작, 반복되는 다중 크롭 작업에 적합합니다.

클립보드 이미지도 바로 붙여넣을 수 있나요?

가능합니다. `Ctrl + V`로 이미지를 바로 가져와 작업을 시작할 수 있습니다.

레이아웃을 다시 사용할 수 있나요?

가능합니다. JSON으로 저장한 크롭 레이아웃을 다시 불러와 재사용할 수 있습니다.

EXE 빌드는 어떻게 하나요?

`build_exe.bat`를 실행하면 의존성 확인부터 빌드까지 자동으로 처리합니다.
