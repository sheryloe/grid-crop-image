# grid-crop-image Service Roadmap

## 서비스 목표

스크린샷을 빠르게 잘라 저장하고 후속 문서 작업까지 자연스럽게 이어지는 이미지 유틸리티로 다듬습니다.

## Priority 1

- 다중 파일 배치 처리와 저장 이름 규칙을 추가합니다.
- Undo/Redo와 작업 세션 저장을 지원합니다.
- 블로그용 캡처 워크플로에 맞춘 기본 프리셋을 제공합니다.

## Priority 2

- OCR 보조 기능을 붙여 문서 캡처 활용도를 높입니다.
- 결과물을 바로 검수할 수 있는 QA 출력이나 요약 뷰를 제공합니다.
- 릴리스 패키징과 소개용 데모 자료를 보강합니다.

## Ops

- 이미지 처리 실패 사례와 복구 방법을 문서화합니다.
- 설정 파일 포맷과 마이그레이션 정책을 정리합니다.
## Blog Automation Note

- Add preset workflows for splitting 2x2 or 3x3 image-prompt results into separate blog assets.
- Position this as a cost-saving image API workflow for blog automation and traffic-oriented content production.
